About

unForget is a web based to-do list app, built with HTML 5, JavaScript, and CSS by a man who has no idea what he is doing.

Main Features and Documentation

Lists:

unForget supports the creation of multiple lists. You must have at least one list to add tasks. To create a list, navigate to the list creation form and type a name for it. Then, press "enter" on your keyboard or the "new" button to submit the form. To select a different list, click its list link in the list bar. Right click on a list link to open a context menu. Then, options to open, delete, and edit the list will be available.

Tasks:

When a list is selected, tasks can be added to it by typing into the task creation form and submitting it. Tasks can also be marked as complete by clicking on the empty box; they can be marked incomplete by clicking again on the box. By clicking the X and confirming it, a task can be deleted. The order of tasks can be changed by dragging from the handle and dropping it over a pre-existing task. This will move the dragged task to the position of the task it is dropped on.

Menu Bar:

This feature is off be default. It can be enabled from the settings menu. If enabled, use the shortcut buttons on the menu bar (at the top of the page) to save time.

Page Context Menu:

The general context menu can be accessed by right clicking on the page. This will provide usefuln shortcuts. By right clicking on a task list in the list bar, another context menu will open. Learn more about this type of context menu under "Lists".

Due Dates: new

To use due dates, right click on a task and select the due date option on the context menu. A menu will open, and a due date and time can be selected. The task's styling will change depending on how close the due date is. It will be bolded if any date is set, orange if the date is within 72h, red if it is within 24h, and black if it is overdue.

Notifications: new

If enabled, a notification will be sent at a set time every day. It will outline when tasks must be completed and how many are outstanding. A notifications will also be sent, periodically, before a task's due date - 7 days, 3 days, 24 hours, and 3 hours before. At the time a task is due, a notification will also be sent. If a due date is selected without a due time, the notification will come at 00:00.

Settings:

Right clicking on the app page to open the general context menu, and selecting the settings option will open the settings page. This feature is still being worked on.

More Info:

Confirmation Override:

All confirmation alerts can be overridden by holding down the "shift" key when performing the action.
